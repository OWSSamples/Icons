## 🎯 CONSEJOS DE BÚSQUEDA

### **1. Busca por Sinónimos**
- **"Arrow"** → ArrowUp, ArrowDown, ArrowLeft, ArrowRight
- **"Check"** → Check, Tick, Confirm, Success
- **"Close"** → Close, X, Cancel, Dismiss
- **"Settings"** → Settings, Gear, Cog, Configuration

### **2. Usa Prefijos Comunes**
- **"Arrow"** → Para todas las flechas
- **"Icon"** → Para iconos básicos
- **"Simple"** → Para versiones simples
- **"Circle"** → Para iconos en círculo
- **"Square"** → Para iconos en cuadrado

### **3. Busca por Sufijos**
- **"Regular"** → Estilo línea estándar
- **"Filled"** → Estilo relleno sólido
- **"Light"** → Estilo línea fina
- **"Duotone"** → Estilo dos colores
- **"Small"** → Versión pequeña
- **"Circle"** → En círculo
- **"Square"** → En cuadrado

### **4. Usa IntelliSense**
```typescript
// Escribe y usa autocompletado
import { Arrow } from '@opendex-origon/icons';
// ↑ Te mostrará todas las opciones que empiecen con "Arrow"
```
